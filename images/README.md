# 图片目录

这个目录用于存放宠物聚会平台的图片资源：

## 预期的图片文件：
- logo.png - 平台Logo
- hero-bg.jpg - 主视觉背景图
- feature-*.jpg - 功能亮点图片
- place-*.jpg - 推荐地点图片
- user-*.jpg - 用户头像图片

## 图片规格建议：
- Logo: 200x200px, PNG格式，透明背景
- 背景图: 1920x1080px, JPG格式
- 功能图: 400x300px, JPG格式
- 地点图: 400x250px, JPG格式
- 头像: 100x100px, JPG格式

当前使用CSS渐变色作为占位图片。
